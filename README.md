## ⚙️ Setup

1. Fork this repository
2. Add secrets in GitHub Settings:
   - `TELEGRAM_BOT_TOKEN`
   - `NEWS_API_KEY` (optional)
   - `RAZORPAY_LINK`
3. That's it! System runs automatically
